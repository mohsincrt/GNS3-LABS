# GNS3-LABS
This repository contains a collection of my GNS3 networking labs created while studying for CCNA/CCNP. These labs are designed to help understand routing, switching and network troubleshooting through practical hands on configurations.
# Topics Covered
Routing (Static, OSPF, EIGRP)
Switching (VLANs, TRunking, STP)
NAT, ACLs, DHCP
TRoubleshooting exercises
Multi-router topologies
